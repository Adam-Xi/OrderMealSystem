# 项目简介
本项目旨在模拟餐厅自助点餐系统，消费者可通过餐桌上粘贴的点餐二维码进行点餐，后台可以查看用户订单详情并进行相应处理。<br />
本项目为练手小项目，目的为了验证对于Linux网络编程方面的粗浅理解，若有不足，请留言指正，不胜感激！
